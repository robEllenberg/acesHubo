Installing the RTT on Windows

Unpack the RTT (should have something like orocos-rtt-x.x.x/

cd orocos-rtt-x.x.x/
mkdir aces
cp aces/utils/aces-cmakelists orocos-rtt-x.x.x/aces/CMakeLists.txt
cp aces/utils/rtt.cmake oroco-rtt-x.x.x/orocos-rtt.cmake

Make an out of source build directory
Use cmake-gui to generate the VC++ Project
Compile the project in VC++

The VC++ solution contains projects for install & uninstall
Default path is /Program Files/orocos-rtt
